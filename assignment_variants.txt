Варианты для творчества с наследованием и интерфейсами



1. персона: студент, преподаватель, зав. кафедрой
 Возможные поля: Фамилия, имя, отчество, курс, факультет, универ, предмет, научная степень, кафедра...
 Возможные интерфейсы (методы, их группы): уволить, отчислить, выгнать из пары, заплатить зарплату

2. персона: служащий,  рабочий, инженер
 Возможные поля: Фамилия, имя, отчество, цех, разряд, номер кабинета, домашний адрес
 Возможные интерфейсы (методы, их группы): дать задание, заплатить зарплату, заплатить премию, оштрафовать

3. кадры: рабочий, , инженер, администрация
 Возможные поля: Фамилия, имя, отчество, цех, разряд, номер кабинета, домашний адрес
 Возможные интерфейсы (методы, их группы): дать задание, заплатить зарплату, заплатить премию, оштрафовать

4. изделие: узел: деталь, механизм, 
 Возможные поля: название, изготовитель, марка, вес, цена
 Возможные интерфейсы(методы, их группы): проверить комплектацию, проверить работоспособность, заменить

5. организация: страховая компания, судостроительная компания, завод
 Возможные поля: название, объем капитала, чистый доход, адрес, город
 Возможные интерфейсы(методы, их группы): заплатить налоги, оформить страховой полис, оформить договор о поставке

6. печатное издание: журнал, книга, учебник
 Возможные поля: название, издательство, год, город, автор, кол-во страниц
 Возможные интерфейсы (методы, их группы): оформить подписку, купить, почитать

7. испытание: тест, экзамен, выпускной экзамен
 Возможные поля: название, предмет, экзаменатор, кол-во вопросов, оценка
 Возможные интерфейсы (методы, их группы): сдать, пересдать, получить оценку автоматом

8. место: область, город, мегаполис, неопределено
 Возможные поля: название, геогр. координаты, площадь, кол-во жителей, государство, мер, губернатор
 Возможные интерфейсы (методы, их группы): задать координаты, назвать, сменить главу

9. товар: игрушка, продукт, молочный продукт
 Возможные поля: название, вес, изготовитель, дата изготовления, срок годности, 
 Возможные интерфейсы (методы, их группы): Проверить годность, купить, сьесть 

10. документ: квитанция, накладная, чек
 Возможные поля: номер, сумма, продавец, дата
 Возможные интерфейсы (методы, их группы): распечатать, оплатить, получить товар

11. транспортное средство: автомобиль, поезд, экспресс
 Возможные поля: название, номерной знак, номер маршрута, цена билета, 
 Возможные интерфейсы (методы, их группы): купить билет, заправить топливом, управлять

12. двигатель, двигатель внутреннего сгорания, дизель, турбореактивный двигатель
 Возможные поля: марка, объем, мощность, потребление топлива на 100 км, 
 Возможные методы (группы методов, интерфейсы): отрегулировать карбюратор, заменить, 

13. государство: республика, монархия, королевство
 Возможные поля: название, президент, премьер-министр, вождь, король, царь, ...
 Возможные методы (группы методов, интерфейсы): Передать правление, избрать главу государства, импичмент

14. животное: млекопитающие, парнокопытные, птицы
 Возможные поля: название, вес, возраст, летает (да/нет), наличие рогов, цвет шерсти, цвет перьев, цвет
 Возможные методы (группы методов, интерфейсы): покормить, перебинтовать крыло, подоить, 

15. корабль: пароход, парусник, корвет
 Возможные поля: название, кол-во палуб, кол-во парусов, кол-во людей экипажа, кол-во пассажиров
 Возможные методы (группы методов, интерфейсы): Поднять якорь, паруса: спустить, поднять, полный вперед, полный назад

16. Музыкальный инструмент: синтезатор, гитара, акустическая гитара, электрогитара, бубен, барабаны, тарелки...
 Возможные поля: изготовитель, марка, кол-во струн, кол-во клавиш,цена, 
 Возможные методы: настроить, играть, 

17. Недвижимость: квартира, многокомнатная квартира, офис, дом, гараж
 Возможные поля: улица, №дома, №квартиры, кол-во комнат, общая площадь, жилая площадь, площадь приусадебного участка, 
 Возможные методы: арендовать, купить, отремонтировать крышу

18. Мебель: стол, письменный стол, шкаф, шкаф-купе, табуретка, полочка
 Возможные поля(атрибуты): название, модель, габориты, вес, цвет
 Возможные интерфейсы: разбираемый (методы разобрать, собрать), 

19. Маршрут: пеший маршрут, автомобильный маршрут, маршрут на общественном транспорте
 Возможные поля(атрибуты): Длина (км), время движения,
 Возможные интерфейсы: подготовить, пройти, изменить

20. Окна: металлопластиковые окна, деревянные окна, ...
 Возможные поля(атрибуты): габориты высота, ширина, кол-во стекол, изготовитель, 
 Возможные методы: открыть, закрыть, поменять стекла

21. Телефон : мобильный телефон, домашний телефон, радиотелефон, смартфон
 Возможные поля: цена, цвет, марка, кол-во сим-карт, поддерживаемый стандарт(ы) сети, радиус действия, ОС
 Возможные интерфейсы: проверить работоспособность, перепрошить, обновить, 

22. Автомобильный номер: временный номер, служебный номер, зарубежный номер, 
 Возможные поля: серия (геогр. область), номер, буквы, еще что-то
 Возможные интерфейсы: вывести на экран, определить регион, протереть, определить владельца

23. Учебное занятие: урок, лекция, лабораторная, практичка, семинар, экскурсия
 Возможные поля: курс, группа, преподаватель, аудитория, время, длительность
 Возможные интерфейсы: поприсутствовать, получить оценку

24. Созвучие: нота, интервал, аккорд, септаккорд
 Возможные поля: нота1, нота2, нота3, нота4, массив нот, кол-во нот, тональность
 Возможные интерфейсы: определить бас, вывести буквенное обозначение, вывести ноты в канонической последовательности

25. процесс: приложение, служба, вирус, сервер, оконное приложение
 Возможные поля: ид процесса, путь к исполняемому файлу, параметры командной строки при запуске, 
 возможные интерфейсы: убить, изменить приоритет

П.С.: комменты и замечания приветствуются.
